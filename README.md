# Movie App üé¨

**Movie App** is an interactive web application that displays the latest movies and TV shows using the **TMDB API**, with a sleek design and smooth user experience.

---

## Features ‚ú®

* Display movies in a **grid layout** with **hover animations** using Animate.css.
* **Toggleable sidebar** with icons (`fa-align-justify` / `fa-xmark`).
* Movie categories:

  * Now Playing
  * Popular
  * Top Rated
  * Trending
  * Upcoming
* **Live search** by movie title.
* **Interactive contact form** with validation for:

  * Name, Email, Phone, Age, and Password.
  * Show/Hide password functionality.
* Movie rating display using **stars**.
* Fully **responsive design** for all screen sizes.

---
